Usage

Colors are expected to be used via &:extend(.color) within a given LESS declaration. For example: 

html {
    &:extend(.text-color); /* [1] */
    &:extend(.background-color); /* [1] */
}

See all the mixins here. The general form is: .text-color--NAME or .background-color--NAME
